# Alzheimer's Disease Detection Using Deep Learning

## Project Overview
This project detects Alzheimer's disease stages from MRI images using Deep Learning.

## Technologies Used
- Python
- TensorFlow
- Keras
- OpenCV
- NumPy

## Dataset Classes
- Non Demented
- Very Mild Demented
- Mild Demented
- Moderate Demented

## How to Run
1. Install requirements
2. Run app.py
3. Upload MRI image
4. Get prediction
